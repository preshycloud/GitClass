Introduction to Git

Git is an open-source version control system that is used to track the changes made to our code.

Advantages of using Git

  • Fast and efficient
  • Distributed version control
  • Enhanced collaboration
  • Compatibility
  • Data integrity

Configuring Git

git config --global user.name "Aderibigbe Precious"

git config --global user.email "aderibigbeprecious760@gmail.com"

Basic Git command

  • git init: used to initialise a git repository
  • git add . : used to stage changes made to our code
  • git status: used to check the status of our code
  • git commit -m "commit message" : used to save the stage chnages with a descriptive message
  • git remote add origin: used to connect our local repo to our remote repo on github
  • git push origin main: used to push your code from local repo to remote repo on github
